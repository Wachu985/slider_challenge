# slider_challenge

## Project: Slider Challenge

This repository contains an implementation of a challenge inspired by The Flutter Way video:
https://www.youtube.com/watch?v=xqAT-yJTTwM

## Description
- **Objective:** Recreate and integrate an interactive slider with behavior and styling similar to the reference video.
- **Framework:** Flutter (Dart).

## Features
- Custom slider with smooth animations.
- Local state support and real-time visual updates.
- Minimal structure to expand the challenge with additional controls and tests.

## Preview
- Check `lib/main.dart` for the entry point and the main UI.

## Requirements
- Install Flutter: https://docs.flutter.dev/get-started/install
- Use a Flutter SDK compatible with the stable channel.

## How to Run
1. Open a terminal at the project root.
2. Get dependencies:

```bash
flutter pub get
```

3. Run the app on a device or emulator:

```bash
flutter run
```

## Project Structure
- `lib/`: Flutter source code; main file is [lib/main.dart](lib/main.dart).
- `android/`, `ios/`, `macos/`, `web/`, `windows/`, `linux/`: platform folders generated by Flutter.
